Mysql数据库远程机器无法连接，用户名密码正确依旧出现如下提示：

        access denied for user 'root'@'localhost'(using password:YES)

此问题几乎可以肯定是权限问题，解决方案如下（环境为CentOS 6）：
<!-- more -->


``` ruby 
class Fixnum
  def prime?
    ('1' * self) !~ /^1?$|^(11+?)\1+$/
  end
end
```
